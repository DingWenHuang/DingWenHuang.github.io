MERN 專案-學生與教職員課程系統

By DingWenHuang 09 June 2023

 

連結：https://mern-practice-client.onrender.com/

(由於部屬在免費的 render.com 上，首次進入需花費約一分鐘等待服務啟動)

github client 端：https://github.com/DingWenHuang/mern-client0716

github server 端：https://github.com/DingWenHuang/mern-server0716

 

專案簡介：

這是一個學生與教職員課程系統，使用者可以申請帳號，並選擇身分，根據身分的不同，可以進行不同的操作。

 

使用技術：

  • 前端：React
  • 後端：Express.js
  • 資料庫：MongoDB
  • 驗證方式：JWT

 

實作內容：

所有使用者皆可使用註冊、登入、登出功能

並且可以在個人頁面更改使用者名稱、信箱、密碼

講師可以開設新課程，並在課程頁面中對課程進行修改與刪除

學生則可以搜尋課程並註冊，同時可以在課程頁面找到已註冊課程，也可以做退選課程操作

 

介面截圖與解說：

1.登入與註冊中，使用 joi 套件進行簡易的欄位格式驗證並回傳錯誤訊息，並且將密碼以 bcrypt 雜湊處理後儲存到資料庫內

2.個人頁面中，可以看到用戶自己的個人資料，也可以進行更新操作與變更密碼 更新資料時會先帶入舊有的資料提供使用者直接修改，並使用密碼驗證確定為使用者本人做修改，同時修改信箱時會檢查信箱是否已經由其他使用者註冊過 更新密碼則需要輸入正確的舊密碼，並且新密碼的部分需要輸入兩次作為確認才能進行密碼變更

3.講師的課程頁面中可以看到自己開設的課程，也可對課程進行更新與刪除 更新操作一樣會先帶入舊有的課程資料提供講師做修改 刪除操作時會先跳出確認式窗後才進行刪除

4.講師的新增課程頁面可以提供開設新課程，新增後會導向回課程頁面，就可以看到新增的課程了

5.學生的課程頁面則可以看到註冊過的課程，並且可以進行退選操作

6.學生也可以進行搜尋課程來做註冊，這邊有做簡單的快取機制，只會在首次進入搜尋課程路徑時呼叫 API，接著將回傳的資料儲存在瀏覽器上 排序可以根據課程新增時間、學生人數多寡、課程價格高低進行升序與絳序排序 搜尋功能，也實做了簡單的模糊搜尋功能，並且在輸入時會即時更新清單內容
