作品集簡介：

 

1.MERN 專案-學生與教職員課程系統

連結：https://mern-practice-client.onrender.com/

(因使用免費的 render.com 進行部屬，首次進入需等待約一分鐘以啟動服務)

GitHub 客戶端：https://github.com/DingWenHuang/mern-client0716

GitHub 伺服器：https://github.com/DingWenHuang/mern-server0716

 

專案簡介：

這個專案是一個學生與教職員課程系統，使用者可以註冊帳號並選擇身分。根據不同的身分，使用者可以執行不同的操作。

 

使用技術：

  • 前端：React
  • 後端：Express.js
  • 資料庫：MongoDB
  • 驗證方式：JWT

 

實作內容：

  • 提供使用者註冊、登入、登出功能

  • 在個人頁面中，使用者可以更改名稱、信箱和密碼

  • 講師可以建立新課程，同時在課程頁面中進行修改和刪除操作

  • 學生可以搜尋課程並進行註冊，同時可以在課程頁面查看已註冊課程，並進行退選操作

詳細內容與流程圖請參考：https://dingwenhuang.github.io/blog/portfolio_1/

 

2.Spring Boot 專案 - 電商平台後端

連結：https://springboot-mall.onrender.com/

(因使用免費的 render.com 進行部屬，首次進入需等待約一分鐘以啟動服務)

GitHub：https://github.com/DingWenHuang/springboot-mall-deploy

 

專案簡介：

這個專案是一個使用 Spring Boot 架構的電商平台後端。主要功能涵蓋商品管理、使用者管理和訂單管理。同時，進行了單元測試以確保 API 的可靠性。

 

使用技術：

  • 後端：Spring Boot
  • 資料庫：MySQL

 

實作內容：

  • 商品管理的 CRUD 操作

  • 使用者管理的註冊和登入功能

  • 訂單管理的新增和查看歷史訂單功能

  • 為上述功能添加單元測試

詳細內容與流程圖請參考：https://dingwenhuang.github.io/blog/portfolio_2/
