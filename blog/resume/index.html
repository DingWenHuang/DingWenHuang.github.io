作品集簡介：

 

1.MERN 專案-學生與教職員課程系統

連結：https://mern-practice-client.onrender.com/

(由於部屬在免費的 render.com 上，首次進入需花費約一分鐘等待服務啟動)

github client 端：https://github.com/DingWenHuang/mern-client0716

github server 端：https://github.com/DingWenHuang/mern-server0716

 

專案簡介：

這是一個學生與教職員課程系統，使用者可以申請帳號，並選擇身分，根據身分的不同，可以進行不同的操作。

 

使用技術：

  • 前端：React
  • 後端：Express.js
  • 資料庫：MongoDB
  • 驗證方式：JWT

 

實作內容：

所有使用者皆可使用註冊、登入、登出功能

並且可以在個人頁面更改使用者名稱、信箱、密碼

講師可以開設新課程，並在課程頁面中對課程進行修改與刪除

學生則可以搜尋課程並註冊，同時可以在課程頁面找到已註冊課程，也可以做退選課程操作

詳細內容與介面截圖請參考：https://dingwenhuang.github.io/blog/portfolio_1/

 

2.Spring boot 專案-電商平台後端

連結：https://springboot-mall.onrender.com/

(由於部屬在免費的 render.com 上，首次進入需花費約一分鐘等待服務啟動)

github：https://github.com/DingWenHuang/springboot-mall-deploy

 

專案簡介：

這是一個 Spring boot 架構的店商平台後端，主要有三大功能，商品功能、使用者功能、訂單功能，同時有加上單元測試去確保 API 的可靠性

 

使用技術：

  • 後端：Spring boot
  • 資料庫：MySQL

 

實作內容：

商品功能的 CRUD

使用者功能的註冊與登入

訂單功能的新增與查看歷史訂單

以上功能個別的單元測試

詳細內容與介面截圖請參考：https://dingwenhuang.github.io/blog/portfolio_2/
