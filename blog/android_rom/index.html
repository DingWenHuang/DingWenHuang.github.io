心情記事 - 手機變磚搶救記

By DingWenHuang 01 August 2023

這陣子常在反問自己，為什麼會突然想轉職走軟體這條路呢?

雖然主要是因為疫情的這段時間中，我一直在想像將來生活的樣貌

有什麼樣的工作能夠讓我能夠跟家人有更多的相處時間?

想著想著又想起以前曾經碰過的一件小趣事

想說既然都寫個人部落格了，除了當面試用的專案展示以外

偶爾記錄些小趣事也不錯，未來如果能發展成技術部落格也是一個職涯發展目標

 

雖說是小趣事，但遇到的當下真的是心想完蛋了…

事情是發生在交大念碩班的時候

那時的 android 手機很流行刷第三方的 ROM

為的就是比官方釋出的版本更快體驗到新的 android 系統

有的也會自製許多小工具或更好看的 UI 設計風格

 

我印象中應該是要從 4.4 升 5.0 的時候

當時原廠為了防範安裝第三方 ROM 都會有自己的 boot loader

所以刷 ROM 的第一步就是要先裝第三方的 boot loader 才能載入第三方的 ROM

印象中當時主流有兩個，一個忘記名子了，一個好像是三星比較常用的 wiz 開頭之類的名稱

反正裝好第三方的 boot loader 後就可以載入第三方的 ROM

 

後來陸續刷了幾個不同的 ROM 想說嚐嚐鮮，結果有一次刷完後就無法開機了

連進入到 boot loader 的界面都無法，整個畫面都是白的

我重複開機關機，甚至拔下電池都試過好幾輪了，就是沒用

當下我真的是超緊張，因為本來晚上 10 點開始刷想說 12 點前收工

結果這下好笑了，隔天還要去學校上課，早上沒手機用是要怎麼聯絡

只好一直爬文找方法，論壇上的資源，各種英文關鍵字都試過，快要放棄時

忽然看到一篇變成磚的文章，雖然這篇沒直接解決我的問題，但提供我一個方向

就是先刷回原廠 ROM!!!

 

疑?但現在手機連 boot loader 都進不去要怎麼載入 ROM

而且原廠 ROM 還要先載入原廠的 boot loader，印象中當時是無法取得的

最後是透過 USB 所謂的"線刷模式"就是直接透過 USB 連接到電腦，從電腦下指令去安裝 ROM 的方式

結果我插上 USB 的瞬間心都涼了，連手機都抓不到…..

當時原廠有提供一個線刷工具，正常的用途是救援原廠手機，恢復成出場模式

正常來說插上 USB 連線後，可以辨識出手機的型號跟版本，然後選擇要刷入的原廠 ROM 即可恢復系統

但我的畫面完全沒抓到這台手機，於是我不死心又去系統管理員找，確實有看到 USB 裝置有連結

但是無法辨識型號跟驅動，所以我猜應該是我在刷 ROM 的過程中有某些重要的檔案被第三方 ROM 刪除掉了

後來根據型號找來各種驅動程式，一個一個安裝後做測試

那天半夜電腦裝了快 10 個不同來源的驅動程式在做測試，終於有一個能夠讓系統管理員內辨識到 USB 裝置的了

 

這下問題就簡單了，只要能辨識到手機型號，接下來就是用原廠的救援軟體刷入原廠 ROM 就好

然後啟動後馬上就發現我錯了，我忘了我已經是第三方 boot loader 了

當時我記得原廠的 boot loader 只要改第三方以後，就無法再改回去了

結果就只能放棄刷回原廠的作法了

但我又想到，那既然現在我能夠辨識手機型號了，那去找用"線刷模式"載入第三方 ROM 的程式不就好了嗎?

當時我記得應該已經凌晨一點多了，我又在國外論壇到處搜尋可用的資源

最後找到一個是透過 CLI 指令的方式，去指定載入電腦中設定路徑下的 ROM

當然那時的我甚至不懂那些指令是在幹嘛，比較有印象的是有個 sideload 指令就是了

總之透過了這一翻操作後，終於救回來了!!!!!

再次開機進入到畫面的時候，感動到眼淚真的是差點掉下來

就這樣手機變磚的事件就此落幕，謝謝當年那個沒放棄搶救手機的自己

 

後記：

有了這次的經驗，後來就不太碰刷機了

一方面是因為電子支付跟指紋驗證這類的安全性設定出來後

只要刷入第三方 ROM 基本上那些電子支付方式都會無法使用

有許多第三方 ROM 強調有破解原廠限制，可以正常使用支付功能

但想想信用卡資料等等的內容都可能暴露在風險中，一般人應該也不會特別去刷第三方 ROM 了

 

另一方面是隨著近幾年硬體跟軟體的進步，第三方 ROM 也越來越沒優勢了

早期第三方 ROM 許多主打著"瘦身"特色，讓 android 手機運作更快速

這是因為當時的 android 手機許多中低階手機硬體效能很差，原廠的系統優化又不好

使用原廠 ROM 常常會有卡頓的問題，而且又缺乏第三方 ROM 能提供的漂亮 UI 跟各種好用小工具

才會造就當年第三方 ROM 在社群的風行

 

如今的手機效能都相當夠用，甚至已經有種效能過剩的感覺

除了玩手遊的客群可能會要求更高的硬體效能外

現在安卓的中階機種對於大部分人的使用應該已經綽綽有餘了

 

回想起當年為了追求高 CP 值，想要在便宜的手機上獲得更流暢的效能

同時玩玩新的系統功能或體驗漂亮的客製化界面而跑去刷機

這樣的經驗真的是學生時期一個蠻重要的回憶
