Spring boot 專案-電商平台後端

By DingWenHuang 09 June 2023

 

2.Spring boot 專案-電商平台後端

連結：https://springboot-mall.onrender.com/

(由於部屬在免費的 render.com 上，首次進入需花費約一分鐘等待服務啟動)

github：https://github.com/DingWenHuang/springboot-mall-deploy

 

專案簡介：

這是一個 Spring boot 架構的店商平台後端，主要有三大功能，商品功能、使用者功能、訂單功能，同時有加上單元測試去確保 API 的可靠性

 

使用技術：

  • 後端：Spring boot
  • 資料庫：MySQL

 

實作內容：

商品功能的 CRUD

使用者功能的註冊與登入

訂單功能的新增與查看歷史訂單

以上功能個別的單元測試
