Spring Boot 專案-電商平台後端

By DingWenHuang 09 June 2023

 

2.Spring boot 專案-電商平台後端

連結：https://springboot-mall.onrender.com/

(因使用免費的 render.com 進行部屬，首次進入需等待約一分鐘以啟動服務)

GitHub：https://github.com/DingWenHuang/springboot-mall-deploy

 

專案簡介：

這個專案是一個使用 Spring Boot 架構的電商平台後端。主要功能涵蓋商品管理、使用者管理和訂單管理。同時，進行了單元測試以確保 API 的可靠性。

 

使用技術：

  • 後端：Spring Boot
  • 資料庫：MySQL

 

實作內容：

  • 商品管理的 CRUD 操作

  • 使用者管理的註冊和登入功能

  • 訂單管理的新增和查看歷史訂單功能

  • 為上述功能添加單元測試

 

流程圖：

  1. 使用者功能 - 註冊

  1. 使用者功能 - 登入

  2. 訂單功能 - 新增訂單

  3. 訂單功能 - 取得訂單

  4. 商品功能 - 新增商品

  5. 商品功能 - 取得商品

  6. 商品功能 - 修改商品

  7. 商品功能 - 刪除商品

  8. 商品功能 - 取得所有商品
